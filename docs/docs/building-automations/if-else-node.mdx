## If Else Node

Use `If Else Node` for conditional execution based on a condition string. 

| Parameter           | Description                                                                                                              |
|---------------------|--------------------------------------------------------------------------------------------------------------------------|
| `condition`         | Condition string to evaluate.                                                                                           |
| `if_nodes`          | List of `action_node` or `if_else_node` nodes.                                                                           |
| `else_nodes`        | List of `action_node` or `if_else_node` nodes.                                                                           |

```json
{
    "condition": "has_captcha",
    "if_nodes": [
        // Actions to execute if condition is true
        {
            "interaction_action": {
                // ... action definition
            }
        }
    ],
    "else_nodes": [
        // Actions to execute if condition is false
        {
            "interaction_action": {
                // ... action definition
            }
        }
    ]
}
```

Condiiton is a python like expresstion that is evaluated to a boolean value. It can use variables from `input_parameters`, `generated_parameters` and `secure_parameters`.

```json
{
    "condition": "{variable_name[0]} is not None and {variable_name[0]} != 'random_value'",
}
```

### Example

```json
{
  "type": "if_else",
  "condition": "has_captcha",
  "if_nodes": [
    {
      "type": "action",
      "interaction_action": {
        "agentic_task": {
          "task": "Solve the captcha",
          "max_steps": 10,
          "backend": "browser_use"
        }
      }
    }
  ],
  "else_nodes": [
    {
      "type": "action",
      "interaction_action": {
        "click_element": {
          "command": "get_by_role(\"button\", name=\"Continue\")",
          "prompt_instructions": "Click continue"
        }
      }
    }
  ]
}
```

### Else Nodes Are Optional

If you only need to act when a condition is true:

```json
{
  "type": "if_else",
  "condition": "needs_verification",
  "if_nodes": [
    {
      "type": "action",
      "action": {}
    }
  ],
  "else_nodes": []
}
```
